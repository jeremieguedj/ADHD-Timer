Privacy Policy

ADHD Netflix Timer — Last updated: March 11, 2026

In short: This extension collects no personal data, has no analytics, and sends nothing to external servers. Everything stays on your device.

What Data We Collect

None. ADHD Netflix Timer does not collect, transmit, or store any personal information, browsing history, or usage data.

Local Storage

The extension uses chrome.storage.local to save your preferences on your own device. This includes:

  • Timer settings (duration, mode, episode count)
  • Bunny character name
  • Suggested next activity text
  • Sound preference (on/off)
  • Active timer state (so the timer survives page refreshes)

This data never leaves your browser. It is not synced, uploaded, or shared with anyone.

Netflix Host Permission

The extension requests access to https://www.netflix.com/* solely to display the visual timer overlay and bunny character on Netflix video pages. It does not read, modify, or access any Netflix account information, viewing history, or personal data.

Third-Party Services

This extension does not use any third-party services, analytics tools, advertising networks, or external APIs. There are no network requests made by this extension.

Children's Privacy

This extension is designed to be used by parents to manage children's screen time. Since no data is collected at all, there are no children's privacy concerns. The extension complies with COPPA by design — it simply cannot collect data because it has no mechanism to do so.

Changes to This Policy

If this privacy policy is ever updated, the changes will be noted here with a new "Last updated" date. Since the extension collects no data, significant changes are unlikely.

Contact

If you have questions about this privacy policy, you can open an issue on the extension's support page in the Chrome Web Store.
